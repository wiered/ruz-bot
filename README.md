# RuzMSTUCA Bot
Бот-помощник для просмотра расписания МГТУ ГА

## Docker build
If you want to run this with docker:

1. Clone repo
```bash
git clone https://github.com/wiered/ruz-bot.git
```

4. Cd to ruz-bot folder
```bash
cd ruz-bot
```

3. Build Docker image
```bash
docker build -t ruz-bot .
```

4. Run Docker image
```bash
docker run ruz-bot
```
